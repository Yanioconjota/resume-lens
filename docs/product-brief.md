# Product Brief

## Vision

ResumeLens helps engineers improve their resumes by providing AI-powered analysis tailored to specific target roles.

## Target Users

- Software engineers looking for new roles
- Engineers transitioning between specializations (e.g., Angular to React)
- Job seekers wanting actionable resume feedback

## MVP Scope

### Core Features

1. **Target Role Selection**
   - Senior Frontend Engineer
   - React / Next.js Engineer
   - Angular Engineer
   - Full Stack JavaScript Engineer

2. **Resume Upload**
   - PDF, DOCX, TXT formats
   - Max 5MB file size
   - Client-side validation

3. **AI Analysis**
   - Parse resume into structured JSON
   - Evaluate against target role
   - Generate strengths, issues, rewrites

4. **Results Display**
   - Overview with quick stats
   - Parsed resume view
   - Issue list with severity
   - Before/after rewrites
   - Format recommendations

### Out of Scope (MVP)

- User authentication
- Database storage
- Analysis history
- Export to PDF/DOCX
- OCR for scanned documents
- Multiple languages
- Payment/subscriptions

## Success Criteria

1. User can complete full flow in under 60 seconds
2. Analysis results are actionable and specific
3. UI is clean, professional, and portfolio-worthy
4. Code demonstrates strong frontend architecture
